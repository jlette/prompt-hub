# prompt-hub
Une plateforme pour découvrir, créer et partager des prompts.

Ce projet a été développé dans le cadre d'un apprentissage approfondi des standards modernes d'Angular. Il met en pratique des concepts avancés tels que l'architecture Standalone, les Signals, le mode Zoneless, la gestion complexe des formulaires, le routing, les guards, et l'authentification, le tout connecté à une API REST (NestJS).

---

## 🚀 Installation et Lancement

### 1. Prérequis (Backend)
L'application nécessite une API pour fonctionner. Assurez-vous d'avoir téléchargé et lancé le projet `prompt-hub-backend` (NestJS). Il doit impérativement tourner sur le port `3000`.

### 2. Démarrer l'application Angular
Une fois le dépôt cloné, placez-vous à la racine du projet dans votre terminal et exécutez les commandes suivantes :

```bash
# Installation des dépendances
npm install

# Lancement du serveur de développement
npm start:auth
